# Tool for Analytics

Teaching notes, modules, tutorials etc. for the 'Tools for Analytics' course which ran until Fall 2025. 

The course notes are automatically generated using GitHub Actions when push to GitHub and can be seen at https://bss-osca.github.io/tfa/.
